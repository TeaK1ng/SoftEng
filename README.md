## Лабораторная работа №1
### Выведите в консоль три строки. Первая – любое число. Вторая – любое число в виде строки. Третья – любое число с плавающей точкой.

```python
print(123)
print('123')
print(1.23)
```
### Результат.
![1](https://github.com/TeaK1ng/SoftEng/blob/Tema_2/lab_2/1.jpg)

## Выводы

В данном коде выводятся три строки с использованием функции `print()`. Каждая строка содержит разные значения:

1. `print(123)`: Выводится целое число 123. Это число не взаимодействует со строковыми операциями и выводится как есть.

2. `print('123')`: Выводится строка '123', так как она заключена в одинарные кавычки. В этом случае это текстовая строка, а не число.

3. `print(1.23)`: Выводится число с плавающей точкой 1.23. Так же, как и в первом случае, оно выводится как числовое значение.

## Лабораторная работа №3
### Выведите в консоль три строки. Первая – обычная строка. Вторая – F строка с использованием заранее объявленной переменной. Третья – сложите две или более строк в одну.

```python
print('Привет, Мир!')
world = "Мир"
print(f"Привет, {world}!")
one = 'Привет, '
two = 'Мир!'
print (one + two)
```
### Результат.
![3](https://github.com/TeaK1ng/SoftEng/blob/Tema_2/lab_2/3.jpg)

## Выводы
В данном коде совершаются различные действия со строками

1. `print('Привет, Мир!')`: простой элемент, который выводит в консоль текст, который указан в одинарных кавычках.

2. `print(f"Привет, {world}!")`: в этом элементе используются f-строка, которая берёт переменную со значением, которые есть в текущей области видимости, и подставляет его в строку. В самой строке нужно указать переменную в фигурных скобках.

3. `print (one + two)`: в данном элементе кода складываются две переменные, которые хранят в себе строковые значения. При их сложении получается единая строка.


## Лабораторная работа №4
### Выведите в консоль три строки. Первая – трансформация любого типа переменной в bool. Вторая – трансформация любого типа переменной в float или int. Третья – трансформация любого типа переменной в str.

```python

two = 142

```
### Результат.

## Выводы

2. `print(float(two))`: в данном элементе переменная типа int трансформируется в float и из-за этого у числа 142 появляется плавающая точка 142.0

3. `print(str(three))`: в данном элементе переменная является пустой(не имеет значения, None), но при преобразовании её в строку "описание пустоты None" становится текстом.

## Лабораторная работа №6
### Создайте две любые числовые переменные и выполните над ними несколько математических операций: возведение в степень, обычное деление, целочисленное деление, нахождение остатка от деления. При желании вы можете проверить как работают эти вычисления с разными типами данных, например, сначала создать две переменные int, затем создать две переменные float и наконец создать переменные типа int и float и провести над ними операции, прописанные выше.

```python
a = 12
b = 5
print('Возведение в степень', a ** b)
print('Обычное деление', a / b)
print('Целочисленное деление', a // b)
print('Нахождение остатка от деление', a % b)
```
### Результат.
![6](https://github.com/TeaK1ng/SoftEng/blob/Tema_2/lab_2/6.jpg)

## Выводы
В данном коде над двумя переменными совершаются математические операции

1. `a ** b`: возводит значение a в степень значения b.

2. `a / b`: делит значение a на значение b

3. `a // b`: делит значение a на значение b без дробной части

4. `a % b`: находит остаток при делении значения a на значение b
